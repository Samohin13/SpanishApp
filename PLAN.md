## ✅ Готово

- [x] **Flashcards** — карточки с flip-анимацией, SM-2 алгоритм, выбор уровня/категории
- [x] **Dictionary** — словарь с поиском, фильтры по уровню и типу слова
- [x] **Weak Words** — слабые слова (точность < 60%), кнопка тренировать
- [x] **Conjugation** — спряжения 20+ глаголов × 6 времён, раскрываемые таблицы
- [x] **AI Chat** — чат с Claude-репетитором, коррекция ошибок, TTS ответов
- [x] **Achievements** — 17 достижений, разблокированные/заблокированные
- [x] **Profile** — XP прогресс, стрик, статистика, переход к ачивкам
- [x] **HomeScreen** — дашборд, стрик, слово дня, план дня, XP бар
- [x] **TTS** — озвучка испанских слов (Android TTS, es-ES)
- [x] **STT** — распознавание речи (базовый, для будущего Pronunciation)
